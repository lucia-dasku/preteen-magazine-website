# Tween Magazine Website

A responsive multi-page magazine-style website designed and developed as part of a Web Development coursework project.

## Project Overview
This project involved designing and building a fully responsive website for a digital magazine. The goal was to simulate a real-world scenario of creating a content-driven website with multiple pages, structured navigation, and integrated media.

The website includes sections such as lifestyle, horoscope, and newsletter, targeting a young audience.

## Features
- Multi-page website (5 pages) with structured navigation
- Responsive design using Bootstrap
- Custom CSS styling for visual identity
- Integration of media (images and video)
- Basic JavaScript for interactivity
- Accessible design considerations (alt text, structure)

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Learning Outcomes
Through this project, I developed skills in:
- Structuring and organizing multi-page websites
- Responsive design principles
- UI/UX design for content-heavy platforms
- Working with front-end frameworks (Bootstrap)
- Integrating media and improving accessibility

## Coursework Context
This project was developed in 2022-2023 as part of a Web Development module, where the objective was to build a responsive magazine or newsletter website with multiple pages, media integration, and attention to usability and accessibility.

## Future Improvements
